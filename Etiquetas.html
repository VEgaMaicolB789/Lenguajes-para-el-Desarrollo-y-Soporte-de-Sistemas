El elemento HTML representa un rango de texto agregado a un documento. También puede usarlo para representar un rango de texto eliminado. El elemento HTML representa un fragmento de texto en línea que denota la entrada textual del usuario mediante un teclado, entrada de voz o cualquier otro dispositivo de entrada de texto.
